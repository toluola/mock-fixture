# Mock Premier League API

This API serves the latest scores of fixtures of matches in a “Mock Premier League”.

## Project Setup

* Clone this repo
* Run `bundle install`
* Run `rails db:create`
* Run `rails db:migrate`
* Run `rails server`
* Run `redis-server`

## Ruby version
* 2.7.1

## Database
* postgresql
